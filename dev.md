# Dev
## Run the hot compiler
```sh
npm run watch
npm run webpack-w
```
## Debugger extension
- open current project
- run vscode launch (in current workplace, press (`F5`) keyboard).

# Publishing
```sh
npm install -g vsce
vsce login myxvisual
vsce publish
```
